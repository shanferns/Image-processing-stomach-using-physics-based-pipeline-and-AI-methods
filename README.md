# Image-processing-stomach-using-physics-based-pipeline-and-AI-methods
The pipeline segments the organ (stomach: 4D which is 3D + time MRI image) from the rest of the abdomen using two methods: a physics based pipeline and and AI method (using a modified UNet)..The dataset for this project is borrowed from https://sparc.science/datasets/258?datasetDetailsTab=cite

Lu, K.-H., Mosier, K., Liu, Z., Wo, J., Gupta, A., Jarrett, M., Putzke, T., Jaffey, D., Cao, J., Wang, X., Rajwa, B., & Powley, T. L. (2023). 4D upper gastrointestinal magnetic resonance imaging in healthy human subjects and gastroparetic patients (Version 1) [Dataset]. SPARC Portal. https://doi.org/10.26275/23JE-UTE3
